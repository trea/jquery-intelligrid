Welcome to the jQuery Intelligrid Demo

If you're like me, you hate having action buttons on a table/list for your CRUD actions. Intelligrid aims to fix this issue by creating a central actions toolbar and allowing you to select items upon which to act. These actions can direct you to a URL, or perform scripted functions.
